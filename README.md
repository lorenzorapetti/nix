# Nix Configuration

This is my attempt at configuring my computers with [Nix Flakes](https://nixos.wiki/wiki/Flakes).

For an introduction to Flakes, you can watch [this video](https://www.youtube.com/watch?v=JCeYq72Sko0).

## Installing Nix

First of all, you need to install the Nix package manager. You can go to https://nixos.org/download/ or follow along.

#### Linux

```sh
sh <(curl -L https://nixos.org/nix/install) --daemon
```

#### MacOS

```sh
sh <(curl -L https://nixos.org/nix/install)
```
